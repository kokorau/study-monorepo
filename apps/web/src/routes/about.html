📖 About This Project

注目: このページは外部ライブラリに依存しない、プレーンなHTMLページです。 ReactやVueなどのフレームワークを使用せず、純粋なHTML + CSS + JavaScriptで構築されています。

🎯 プロジェクトの目的

Study Monorepoは、現代的なWebアプリケーション開発における モノレポ構成の学習と実践を目的としています。

📚 学習内容

  • pnpm workspaces - 効率的なパッケージ管理
  • Vite - 高速ビルドツールの活用
  • TypeScript - 型安全な開発環境
  • マルチフレームワーク - React/Vueの共存
  • 共通ライブラリ - 認証機能の共有
  • 動的ルーティング - ファイルベースのページ生成

🏗️ アーキテクチャの特徴

このプロジェクトでは、フレームワークベースのアプリケーションと プレーンなHTMLページが共存しています。これにより、用途に応じて 最適な技術選択ができることを実証しています。

  • フレームワークページ - 複雑なインタラクションが必要な場合
  • プレーンHTML - シンプルな情報表示やランディングページ
